# JavaScript Calculator

This is a calculator made entirely with pure JavaScript, CSS and HTML (no frameworks).


## Screenshot
![2021-01-18_15h10_16](https://user-images.githubusercontent.com/59721315/104954065-5d518980-59a6-11eb-836f-a498432fee69.png)

## Features
* Basic math operations (addition, subtraction, multiplication and division)
* Combined operations
* Floating point numbers support
* Backspace button
